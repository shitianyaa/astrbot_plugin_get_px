# 01 · 浅蓝

正式上线模板。直接查看 `preview.png`，或运行构建脚本生成 `preview.html`。

## 视觉方向

- 主题名：**浅蓝**
- 参考气质：《崩坏：星穹铁道》列车票券 — 穹轨极光、轨道线圈、金黄铜线与毛玻璃材质
- 画布：`960 × 540`
- 核心识别：主题票章、旅行日志星轨暗纹和底部钱包票根。
- 钱包票根展示累计旅程、信用余额、背景更新价格和主题拥有状态。
- 功能分区：
  1. **左侧车票存根** — 列车名称、票券类型、乘客头像与昵称、出发日期、当前等级与信用码。
  2. **右侧车票主体** — 包含出发地点与车次指示、今日签到标题、全息投射寄语播放器、每日补给（信用点、里程数、常驻活跃）、列车展望窗（立绘）以及成长账本（累计天数、余额与开拓里程）。
