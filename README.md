# Pixiv 发图

AstrBot 插件 — 通过标签搜索 Pixiv 插画、发送图片并使用每日签到。

## 功能

- 🔍 按标签搜索 Pixiv 插画并发送图片
- 📊 支持 8 种排行榜（今日/周/月/男性向/女性向/原创/新人/漫画）
- 🔎 作品详情查看（ID/标题/作者/标签/浏览量/收藏数）
- 📥 通过作品ID下载并发送图片（支持多页作品指定页码）
- 🔒 R18 过滤（仅非R18 / 仅R18 / 混合）
- ⏱️ 请求频率限制（防刷屏）
- 当天去重（同群/私聊同标签或排行榜按北京时间当天尽量不重复，重启后仍有效）
- 🌐 支持 HTTP/SOCKS5 代理
- 📄 多页作品支持
- 🎯 自然语言自动触发（「来一份图」→ 自动搜图发图）
- 🤖 AI 识图评论（概率触发，支持自定义提示词）
- 每日签到（HTML 卡片、Pixiv/自定义背景、金币、好感度、连续签到和好感度双倍加持）
- WebUI 图片历史：在 AstrBot 插件页面查看已成功发送的 Pixiv 图片记录、本地缩略图和作品黑名单
- 📱 多平台适配（QQ/微信等，非QQ平台自动降级为逐条发送，兼容性请自行测试）
- 🔄 发送失败自动重试（最多3次，递增等待）
- 🧹 临时文件自动清理（发送完成后自动删除，不占用磁盘空间）

## 指令

### 搜索指令

| 指令 | 说明 | 示例 |
|------|------|------|
| `/p [标签] [数量]` | 按标签搜索并发送图片 | `/p 初音ミク 3` |
| `/p [数量]` | 无标签时拉取排行榜 | `/p 5` |

|------|------|------|

| `/prl` | 查看所有排行榜类型 | |
| `/pi <作品ID>` | 查看作品详情 | `/pi 12345678` |
| `/pd <作品ID> [页码]` | 通过作品ID下载并发送图片 | `/pd 12345678` |
| `/签到` | 每日签到并发送签到卡片 | `/签到` |
| `/签到测试` | 管理员预览签到卡片，不写入签到数据 | `/签到测试` |
| `/签到状态` | 查看累计签到、连续签到、金币、好感度和加持状态 | |
| `/签到商店` | 查看好感度加持商品 | |
| `/购买加持 <天数>` | 购买好感度双倍加持，天数支持 1/3/7 | `/购买加持 3` |

### 自然语言触发（需在配置中开启）

| 触发语 | 说明 | 示例 |
|------|------|------|
| `来一份图` | 发送 1 张排行榜图片 | 默认拉取周榜 |
| `来三张初音ミク图` | 搜索标签「初音ミク」发送 3 张 | 支持中文和阿拉伯数字 |
| `来两张萝莉图` | 搜索标签「萝莉」发送 2 张 | 必须以「图」结尾 |
| `来张风景图` | 搜索标签「风景」发送 1 张 | 不带数量默认 1 张 |
| `签到` | 每日签到 | 首次签到发 HTML 卡片；当天重复签到只发文字并扣好感度 |

## 每日签到

签到数据按用户全局统计，同一个用户每天只能成功签到一次。首次签到会获得金币和好感度，连续签到会提高奖励；漏签后连续天数重置为 1，累计签到天数不会清零。

重复签到不会改动累计签到、连续签到、金币或今日奖励记录，只会扣好感度：每次 `-0.20`，当天最多扣 `1.00`，最低扣到 `-10`。

金币商店第一版只出售好感度双倍加持：

| 商品 | 价格 |
|------|------|
| 好感度双倍加持 1 天 | 200 金币 |
| 好感度双倍加持 3 天 | 500 金币 |
| 好感度双倍加持 7 天 | 1000 金币 |

加持购买后立即记录有效期；如果当天已经签到，则从明天签到开始影响收益。重复购买会延长有效期。

好感度等级仅用于展示，不影响奖励：

| 好感度 | 等级 |
|------|------|
| `< 0` | 排斥 |
| `0 - 9.99` | 陌生 |
| `10 - 29.99` | 熟悉 |
| `30 - 69.99` | 亲近 |
| `70 - 139.99` | 信赖 |
| `140+` | 挚友 |

签到卡片固定渲染为 `960x540` 的 16:9 横图。背景可由管理员配置为 Pixiv 每日自动背景或固定本地背景。Pixiv 背景默认优先选择 16:9 横图；如果没有符合比例的候选，会自动回退到未限制比例，避免签到没有背景。Pixiv 背景会沿用 R18 模式、拉黑标签和作品 ID 黑名单；卡片发送成功后会写入图片历史并保留缩略图。下载、记录或 HTML 渲染失败时，签到奖励仍会发放，并回退为纯文字结果。

管理员可使用 `/签到测试` 预览签到卡片和背景效果。该命令不会写入签到数据，也不会写入图片历史，适合调试背景比例和渲染效果。

## WebUI 图片历史

插件会在 AstrBot WebUI 的插件页面中提供「图片历史」页面，用于查看已经成功发送的 Pixiv 图片，包括搜索、排行榜、作品下载和签到 Pixiv 背景。记录内容包括标题、作者、作品 ID、页码、来源、R18 标记、尺寸、发送会话、发送时间、实际发送质量、文件大小和 Pixiv 链接。

图片历史只保存本地缩略图和元数据，不保留原始发送图片；原来的临时图片仍会在发送完成后按原生命周期清理。默认最多保留最近 200 条记录，超过后会同时删除对应旧缩略图。

页面内可将历史作品加入黑名单。加入黑名单时会保留一份缩略图到黑名单目录，并从历史中移除同作品 ID 的记录；黑名单管理页可复制 Pixiv 链接或移除黑名单。移除黑名单只删除黑名单记录和黑名单缩略图，不会恢复原历史记录。

图片历史只用于 WebUI 查看，和当天去重索引相互独立；删除历史记录或加入黑名单不会清空当天已发作品索引。

## 排行榜类型

| 类型 | 含义 | 说明 |
|------|------|------|
| `day` | 今日排行榜 | 每日综合排名 |
| `week` | 本周排行榜（默认） | 每周综合排名，作品质量最高 |
| `month` | 本月排行榜 | 每月综合排名 |
| `day_male` | 男性向日榜 | 男性用户偏好的每日排名 |
| `day_female` | 女性向日榜 | 女性用户偏好的每日排名 |
| `week_original` | 原创周榜 | 无标签/原创作品的每周排名 |
| `week_rookie` | 新人周榜 | 新注册作者的每周排名，发现宝藏的好途径 |
| `day_manga` | 漫画日榜 | 漫画/多页作品的每日排名 |

## 配置项

| 配置 | 说明 | 默认值 |
|------|------|--------|
| `pixiv_refresh_token` | Pixiv refresh_token（必填） | - |
| `pixiv_proxy_url` | 代理地址 | 空（直连） |
| `pixiv_r18` | R18 模式：0=仅非R18，1=仅R18，2=混合 | 0 |
| `filter_manga` | 漫画过滤；开启后普通搜索、普通排行榜和签到背景只要命中 manga 类型就过滤，`day_manga` 漫画日榜保留后门 | 开启 |
| `blacklist_tags` | 拉黑标签，多个标签用英文逗号分隔；精确匹配 Pixiv 标签原文或翻译名，命中的作品不会被搜索、排行、签到背景、详情和下载使用 | furry,裸体,全裸,触手,露出,nsfw |
| `pixiv_ranking_mode` | 默认排行榜类型（无标签时使用） | week |
| `max_count` | 单次最大发送数量（1-20） | 5 |
| `dedupe_ttl_hours` | 大于 0 时启用普通发图当天去重：同群/私聊同标签或排行榜按北京时间当天记录已发作品，重启后仍有效，次日自动清理；设为 0 只关闭普通发图去重 | 24 |
| `request_timeout` | 单张图片下载超时（秒） | 30 |
| `image_quality` | 图片质量：original / large / medium；原图超过阈值自动降级 | original |
| `auto_downgrade_original_mb` | 原图自动降级阈值（MiB），设为 0 禁用 | 3.0 |
| `send_as_forward` | 合并转发发送（聊天记录形式，规避审查） | 开启 |
| `auto_trigger_enabled` | 自然语言自动触发（来份图→搜图发图） | 关闭 |
| `checkin_enabled` | 签到开关 | 开启 |
| `checkin_bot_name` | 签到卡片里的 bot 角色名 | neko |
| `checkin_background_mode` | 签到背景模式：pixiv_daily / custom | pixiv_daily |
| `checkin_background_tag` | 签到 Pixiv 背景标签；多个标签用英文逗号分隔，留空使用默认排行榜 | 空 |
| `checkin_background_aspect_ratio` | 签到 Pixiv 背景图片比例；默认优先 16:9，可填 `2.2:1`、`16:9`、`1:1`、`3:4` 或小数，留空不限制；无匹配候选时自动回退 | 16:9 |
| `checkin_background_aspect_tolerance` | 签到 Pixiv 背景比例容差；`0.25` 表示允许目标比例上下 25% | 0.25 |
| `checkin_custom_background` | 管理员本地固定背景路径，推荐 16:9（1920x1080 或 960x540） | 空 |
| `checkin_avatar_enabled` | 签到卡片显示用户头像 | 开启 |
| `rate_limit_seconds` | 同一用户请求频率限制（秒） | 3 |
| `ai_enabled` | AI 识图开关 | 关闭 |
| `ai_probability` | AI 识图触发概率（0-100%） | 30 |
| `ai_max_images` | AI 识图最大数量 | 3 |
| `ai_pre_message` | AI 识图预回复消息 | 让我先品鉴一番，你稍等喵~ |
| `ai_vision_provider_id` | 识图模型（下拉选择，留空自动） | 空 |
| `ai_comment_provider_id` | 评论模型（下拉选择，留空自动） | 空 |
| `ai_vision_prompt` | 识图提示词 | 描述画风、构图、角色等 |
| `ai_comment_prompt` | 评论提示词（{description}=识图结果） | 根据描述写评论（50字） |

## 获取 Token

使用 [pixiv-token](https://github.com/shitianyaa/pixiv-token) 获取 Pixiv refresh_token。

## 安装

以下方式任选其一：

1. **下载压缩包**：下载本项目为 zip，通过 AstrBot WebUI 插件页的「导入压缩包」安装
2. **粘贴链接**：在 AstrBot WebUI 插件页点击「安装」，粘贴本项目 GitHub 链接
