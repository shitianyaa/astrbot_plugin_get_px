# 指令参考

返回 [README](../../README.md)。

## 发图

| 指令 | 说明 | 示例 |
| --- | --- | --- |
| `/p [标签] [数量]` | 按标签搜索并发送图片，支持含空格的多词标签 | `/p 初音ミク 3` |
| `/p [数量]` | 无标签时随机发送图片 | `/p 5` |

发图默认请求 Lolicon；请求失败或无结果时，有标签回退 Pixiv 搜索，无标签回退 Pixiv 推荐作品。Pixiv 回退仅在配置 `pixiv_refresh_token` 后可用。

## 签到（用户）

| 指令 | 说明 | 示例 |
| --- | --- | --- |
| `/签到` | 每日签到并发送签到卡片 | `/签到` |
| `/签到帮助` | 发送完整签到功能帮助图片 | `/签到帮助` |
| `/签到测试` | 使用当前用户资料和问候配置预览卡片，不写入数据 | `/签到测试` |
| `/签到状态` | 查看累计签到、金币、好感度和加持状态 | `/签到状态` |
| `/签到排行 [今日\|月榜\|连签\|累计]` | 查看当前群独立统计的签到排行 | `/签到排行 月榜` |
| `/签到商店` | 查看加持、背景刷新和主题商品 | `/签到商店` |
| `/购买加持 <天数>` | 购买好感度双倍加持，支持 1/3/7 天 | `/购买加持 3` |
| `/刷新签到背景` | 花费金币重新抽取当天在线背景；一言模式下会同时刷新问候 | `/刷新签到背景` |
| `/签到主题` | 查看当前、已购和可购买主题 | `/签到主题` |
| `/查看主题 <编号>` | 免费查看指定主题的静态预览图 | `/查看主题 1` |
| `/购买主题 <编号>` | 购买主题并自动切换 | `/购买主题 02` |
| `/切换主题 <编号>` | 切换默认或已购买主题 | `/切换主题 默认` |
| `/签到生日` | 查看生日；未保存时自动读取 QQ 资料 | `/签到生日` |
| `/签到生日 设置 MM-DD` | 手动设置签到生日 | `/签到生日 设置 07-11` |
| `/签到生日 清除` | 清除已保存的签到生日 | `/签到生日 清除` |
| `/签到成就` | 查看已解锁成就和下一目标 | `/签到成就` |
| `/签到称号` | 查看已解锁称号 | `/签到称号` |
| `/佩戴称号 <ID或名称>` | 切换卡片称号 | `/佩戴称号 七日同行` |

签到规则、商店与问候等说明见 [签到说明](checkin.md)。

## 签到（管理员）

| 指令 | 说明 | 示例 |
| --- | --- | --- |
| `/签到事件 ...` | 管理员维护年度或单次全局事件 | `/签到事件 添加年度 07-11 相遇纪念日` |
| `/签到导出` | 管理员导出签到完整备份 | `/签到导出` |

## 自然语言触发

开启 `auto_trigger_enabled` 后，可以不带命令前缀触发搜图。

| 触发语 | 效果 |
| --- | --- |
| `来一份图` | 发送 1 张随机图片 |
| `来三张初音ミク图` | 搜索「初音ミク」并发送 3 张 |
| `来两张萝莉图` | 搜索「萝莉」并发送 2 张 |
| `来张风景图` | 搜索「风景」并发送 1 张 |
| `签到` | 触发每日签到 |
