# 指令参考

返回 [README](../../README.md)。

## 发图

| 指令 | 说明 | 示例 |
| --- | --- | --- |
| `/p [标签] [数量]` | 按标签搜索并发送图片，支持含空格的多词标签 | `/p 初音ミク 3` |
| `/p [数量]` | 无标签时随机发送图片 | `/p 5` |

发图默认请求 Lolicon；请求失败或无结果时，有标签回退 Pixiv 搜索，无标签回退 Pixiv 推荐作品。Pixiv 回退仅在配置 `pixiv_refresh_token` 后可用。

普通发图与正式签到背景共享图片去重索引。`dedupe_days` 可设置最近 `0–7` 个北京时间自然日；同群成员共享去重，不同群、私聊用户、标签和图片源相互隔离。Lolicon 图片地址不可用时，可在 `lolicon_image_proxy_origins` 中按行配置多个图片反代 origin；该配置不代理 Lolicon API 或 Pixiv 登录。

## 签到（管理员）

| 指令 | 说明 | 示例 |
| --- | --- | --- |
| `/签到管理 预览` | 使用当前用户资料和问候配置预览卡片，不写入数据 | `/签到管理 预览` |
| `/签到管理 事件 [查看\|列表]` | 查看全局签到事件；动作可省略 | `/签到管理 事件 查看` |
| `/签到管理 事件 添加 <年度\|单次> <日期> <名称>` | 添加年度或单次全局事件 | `/签到管理 事件 添加 年度 07-11 相遇纪念日` |
| `/签到管理 事件 删除 <ID>` | 按事件 ID 删除全局事件 | `/签到管理 事件 删除 1` |
| `/签到管理 导出` | 管理员导出签到完整备份 | `/签到管理 导出` |

只输入 `/签到我的`、`/签到排行`、`/签到商店` 或 `/签到管理` 时，AstrBot 会显示对应层级的文字指令树；使用 `/签到帮助` 发送完整帮助图。文档使用默认前缀 `/`；实际前缀以 AstrBot 的 `wake_prefix` 配置为准。

## 自然语言触发

开启 `auto_trigger_enabled` 后，可以不带命令前缀触发搜图。

| 触发语 | 效果 |
| --- | --- |
| `来一份图` | 发送 1 张随机图片 |
| `来三张初音ミク图` | 搜索「初音ミク」并发送 3 张 |
| `来两张萝莉图` | 搜索「萝莉」并发送 2 张 |
| `来张风景图` | 搜索「风景」并发送 1 张 |
| `签到` | 触发每日签到 |
